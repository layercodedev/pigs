## Broadcast Prompt Feature

Added broadcast prompt (press `b`) to send the same `claude -p` command to all provisioned VMs simultaneously. This is a core feature for parallel agent orchestration — send one task to many agents at once.

Changes:
- `src/types.ts`: Added `'broadcast'` to AppState mode union
- `src/tui.ts`: Added broadcast prompt dialog (yellow border), `b` keybinding, input/cancel handlers, updated status bar
- `src/index.ts`: Added `broadcast-submit` handler that iterates all provisioned VMs, attaches console, and sends `claude -p` command
- `src/__tests__/types.test.ts`: Added test for broadcast mode

## Prompt History Feature

Added prompt history with up/down arrow navigation in prompt and broadcast dialogs. Prompts are persisted to `~/.pigs/history.json` and recalled with arrow keys — essential for iterating on prompts across multiple agent sessions.

Changes:
- `src/prompt-history.ts`: New module — manages prompt history with add, navigate up/down, cursor reset, deduplication, and file persistence (max 100 entries)
- `src/tui.ts`: Added `keypress` handlers on prompt and broadcast textboxes for up/down arrow history navigation, reset cursor on dialog open, added history hint to help screen
- `src/index.ts`: Load history on startup, save prompts on submit (both single and broadcast)
- `src/__tests__/prompt-history.test.ts`: 18 tests covering load, save, add, deduplicate, navigate up/down, draft preservation, cursor reset

## Bulk VM Creation Feature

Added bulk VM creation (press `C` / Shift-C) to create multiple agent VMs at once. Essential for spinning up a fleet of parallel agents quickly — enter a number (1-20) and all VMs are created and provisioned in parallel with live progress updates.

Changes:
- `src/types.ts`: Added `'bulk-create'` to AppState mode union
- `src/tui.ts`: Added bulk create dialog (green border, 50x5), `S-c` keybinding, number input with validation (1-20), submit/cancel handlers. Updated help screen with new keybinding, updated status bar to include `C:bulk-create`. Fixed `q` key to not quit during broadcast/bulk-create modes.
- `src/index.ts`: Added `bulk-create` handler that creates N VMs in parallel via `Promise.all`, then provisions all in parallel with progress tracking (`Provisioned X/N...`)
- `src/__tests__/types.test.ts`: Added test for bulk-create mode

All 114 tests pass. Type checks clean.

## Bulk VM Deletion Feature

Added bulk delete (press `D` / Shift-D) to delete all VMs at once with confirmation. Essential complement to bulk create — tear down an entire fleet of agents in one action instead of deleting one by one. Deletes happen in parallel with live progress, unmounts filesystems and destroys console sessions before deletion.

Changes:
- `src/types.ts`: Added `'confirm-delete-all'` to AppState mode union
- `src/tui.ts`: Added confirm-delete-all dialog (red border, 50x7), `S-d` keybinding, y/n confirmation handlers, updated q/Ctrl-C to dismiss dialog. Updated help screen with `D (shift) Delete ALL VMs at once`. Updated status bar to include `D:delete-all`.
- `src/index.ts`: Added `delete-all` handler that unmounts, destroys consoles, and deletes all VMs in parallel via `Promise.all` with progress tracking (`Deleted X/N...`). Tracks success/failure per VM using a Set.
- `src/__tests__/types.test.ts`: Added test for confirm-delete-all mode

All 115 tests pass. Type checks clean.
